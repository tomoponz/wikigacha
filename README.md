# WikiGacha（v2）

オリジナル実装の「Wikipediaランダム記事×TCG風」デモです。

## 仕様（今回の要望反映）
- **見た目：ダークUI + 黄色アクセント + 中央パック + ミッションパネル**（オリジナルCSSで再構成）
- **パックは 1分で1回復**（最大10）
- **1回のパックで5枚排出**
- **10回ごとに金パック（排出率UP）**
- デイリーミッション（達成で +2 パック）
- 図鑑（キーワード検索 / レアリティ絞り込み）
- バトル（デッキ最大3枚の簡易シミュレーション）
- 🏆 実績（簡易）

## Wikipedia API
- `https://ja.wikipedia.org/api/rest_v1/page/random/summary`

## GitHub Pages
1. このフォルダ内のファイルをリポジトリ直下に配置
2. Settings → Pages → main / root
3. 公開URLへアクセス
